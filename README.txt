BUTTONS:
	Btn_Up     - reset
	Btn_Center - start/ack
	Btn_Left   - move selected column left
	Btn_Right  - move selected column right
	Btn_Down   - drop piece down
	
SSD:
	Displays whose turn it is. Once a player wins, their name flashes. If the game is tied, "tie" is flashed
	
LEDs:
	Displays which state the state machine is currently in
